# Smart Network Discovery & Security Scanner

A Python-based cybersecurity tool designed to discover devices on a local network, identify open ports, detect running services, and generate security-focused reports.

## Features

- Network device discovery
- IP address and hostname detection
- Open port scanning
- Service identification
- User-friendly graphical interface
- PDF report generation
- Scan history management
- Security-oriented network analysis

## Technologies Used

- Python
- Nmap
- CustomTkinter / Tkinter
- SQLite
- ReportLab
- Socket Programming

## How It Works

1. Scan the local network.
2. Discover active devices.
3. Analyze open ports and services.
4. Review results through the dashboard.
5. Generate a professional report.


## Installation

Clone the repository:
git clone https://github.com/amarabadou/Network_Scanner.git

Navigate to the project directory:
cd Network_Scanner

Install dependencies:
pip install -r requirements.txt

Run the application:
python network_scanner.py
